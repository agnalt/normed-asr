# NORMED-ASR demo
A simple Gradio app for exploring the NORMED-ASR dataset. Provides a simple interface for audio and text preview as well as term search and explore functionality.

## Usage
Use `uv` to start the app locally: `uv run dataset_browser.py`

Point `--dataset-dir` to the `final_dataset` directory. The app runs locally. Passing `--share` creates a publically sharable link for the app streaming data from the host.
